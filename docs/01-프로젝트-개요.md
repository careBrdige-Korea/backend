# CareBridge Korea - 프로젝트 개요

## 프로젝트 소개

**"외국인 환자와 국내 병원을 잇는 데이터 기반 의료 매칭 플랫폼"**

CareBridge Korea는 외국인 환자들이 한국의 의료 서비스를 쉽게 이용할 수 있도록 돕는 플랫폼입니다. 사용자가 상담을 신청하면 운영자가 적합한 병원을 추천하고, 사용자의 선택과 예약 결과를 추적합니다.

## 핵심 비즈니스 플로우

```
[상담 접수] → [병원 풀 관리] → [수동 매칭] → [전환 추적]
```

### 1. 상담 접수 (User Input)
- 유저가 웹 폼을 통해 증상, 위치, 언어, 희망 시간을 제출합니다.
- 비회원도 상담 신청이 가능합니다.

### 2. 병원 풀 관리 (Admin DB)
- 운영자가 외국어 응대가 가능한 병원 데이터를 미리 구축합니다.
- 병원 정보: 이름, 지역, 진료과, 외국어 가능 여부

### 3. 수동 매칭 (Admin Process)
- 운영자가 상담 내역을 보고 적합한 병원 1~3곳을 선택하여 유저에게 추천합니다.
- 한 상담 건에 동일한 병원이 중복 추천되지 않도록 제약이 적용됩니다.

### 4. 전환 추적 (Data Tracking)
- 유저가 추천된 병원 중 무엇을 '선택'했는지 기록
- 실제 '예약'이 성공했는지 데이터를 기록
- 추후 매칭 자동화 알고리즘 도입을 위한 데이터 축적

| Database | MySQL |
